# Javadoc




#### ** 7.1 格式化**

>7.1.1 通用格式

Javadoc區塊的基本格式化樣式如下：

    /** 
     * Multiple lines of Javadoc text are written here,
     * wrapped normally... 
     */
     public int method(String p1) { ... }

或是像這個單行格式化樣式的例子：

    /** An especially short bit of Javadoc. */
    
基本格式化樣式是通用的。當沒有使用Javadoc區塊、及Javadoc區塊（包含註解符號）可以使用單行就完成敘述時，單行格式化樣式可替代基本格式化樣式。
